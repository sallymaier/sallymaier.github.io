Off-Register

An experimental jQuery script that brings the imperfection of print to the web.

Printing is an imperfect process. As the number of reproductions increase, its fidelity decreases. Each step in the printing process and each copy creates new chances for error and variation. I explored this by using CMYK screens to reproduce a one-color handwritten phrase to this effect. I still consider myself a novice printer, so each print out of the edition of fifty was unique.

Handwriting does not have spellcheck, either.
